## 文件说明

每个模版目录包含以下关键文件：
- `package.json` - 项目配置和依赖
- `astro.config.mjs` - Astro配置文件
- `tsconfig.json` - TypeScript配置
- `README.md` - 模版说明文档
- `src/pages/index.astro` - 主页面
- `src/layouts/` - 布局组件
- `src/components/` - UI组件
